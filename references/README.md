I kept the partial `vllm/` and `sglang/` shallow clone directories that were created before the TLS disconnects, and downloaded README snapshots for the projects used as architectural references.

- `vllm.README.md` from https://github.com/vllm-project/vllm
- `sglang.README.md` from https://github.com/sgl-project/sglang
- `comfyui.README.md` from https://github.com/Comfy-Org/ComfyUI
- `localai.README.md` from https://github.com/mudler/LocalAI
- `litellm.README.md` from https://github.com/BerriAI/litellm

The project changes avoid copying their code; the references are used for architecture and integration shape only.
